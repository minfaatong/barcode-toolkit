# Barcode Toolkit

- build

```sh
mvn clean install
```

- run cli

```sh
java -jar target/barcode-toolkit-<version>.jar --text 'testing ...' --output 'out.jpg'
```

- run GUI

```sh
java -jar target/barcode-toolkit-<version>.jar --gui
```
